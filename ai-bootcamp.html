AI Syllabus

Session 1: Introduction to Machine Learning

  • What is Machine Learning?
  • Important terminology
  • Types of ML
  • How ML models learn

Session 2: ML Fundamentals

  • Generalization & overfitting
  • Gradient descent
  • Hyperparameters
  • Regularization

Session 3: Classification

  • Logistic regression
  • Normalization
  • Classification thresholds
  • Accuracy, precision, & recall
  • Begin brainstorming final project

Session 4: Artificial Neural Networks

  • What are neural networks?
  • Activation functions
  • Training neural networks

Session 5: Classification with Neural Networks

  • Binary classification
  • Multi-class classification
  • Working with non-numerical data
  • Finalize final project idea

Session 6: Computer Vision

  • What is computer vision?
  • Intro to convolutional Neural Networks
  • Digit classification
  • Collect final project data

Session 7: Natural Language Processing 1

  • Classification with text data
  • Intro to recurrent neural networks
  • Intro to long short term memory neural networks
  • Preprocess final project data & begin training model

Session 8: Natural Language Processing 2

  • Intro to transformer neural networks
  • Bidirectional encoder represeentations from transformers (BERT)
  • Question and answering with BERTs
  • Finish training and fine tuning final project model

Session 9: Closing Ceremony
